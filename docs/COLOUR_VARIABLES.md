# Colour variables – what affects what

1. **Brand** – Top bar on dashboard; background on handset screens 1, 2 & 4 (tap to start, in the queue, thank you).
2. **Waiting** – Waiting buttons on dashboard (column header + “Ready?” buttons).
3. **Ready** – Ready buttons on dashboard (column header + “Collected” buttons); background on handset screen 3 (order ready for collection).
4. **Left column** – Background of left column on dashboard.
5. **Right column** – Background of right column on dashboard.
6. **Middle column** – Background of middle column on dashboard (QR / New order area).
